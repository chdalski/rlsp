# Security Policy

## Reporting a Vulnerability

**Do not file security issues as public GitHub issues.**

Report vulnerabilities using GitHub's private security advisory feature:
**Repository Settings > Security > Advisories > New draft security advisory**

This keeps the details confidential until a fix is available.

## Scope

This policy covers the `rlsp-yaml` binary and its dependencies.

## Response Timeline

- **Acknowledgement**: within 7 days of the report
- **Fix**: within 30 days for confirmed vulnerabilities

We will keep you informed throughout the process and coordinate disclosure timing with you.
