# Open Tasks for rlsp-yaml YAML Language Server

## Completed Features
- [x] LSP bootstrap (tower-lsp, stdio transport)
- [x] YAML parsing with diagnostic publishing
- [x] Hover support
- [x] Document symbols (outline view)
- [x] Structural completion
- [x] Anchor/alias navigation (go-to-definition + find references)
- [x] Folding ranges
- [x] Rename (anchors/aliases)
- [x] Additional validators (unused anchors, flow style, key ordering)
- [x] Document links (URL detection in YAML values)
- [x] Selection ranges (AST-based expand/shrink selection via saphyr MarkedYamlOwned spans)
- [x] Code actions (flow/block conversion, tab fix, unused anchor delete, quoted bool, block scalar)
- [x] Schema support (loading, validation, completion, hover)
- [x] Custom tags (unknown tag validation, modeline + workspace settings configuration)
- [x] Code lens (schema title/URL at top of document)
- [x] On-type formatting (newline auto-indent after colon, block scalar indicators)
- [x] `!include` document links (file path resolution and clickable links)

## Current Test Coverage
- 587 tests passing
- Clippy clean with pedantic + nursery lints

## Architecture Notes
- Pure function design: each feature module exports a pure function (text + position -> result)
- saphyr 0.0.6 for YAML parsing (migrated from yaml-rust2)
- MarkedYamlOwned for AST nodes with span info (used by selection ranges)
- tower-lsp v0.20 with "proposed" features
- tokio for async runtime
